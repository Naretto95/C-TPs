Comment compiler le code :
- dirigez-vous dans le dossier où le fichier .c est placé via le terminal.
- executez la commade : make
- executez la commande : ./"nomdufichier"

Pour générer le doxyfile :
- dirigez-vous dans le dossier où le fichier .c est placé via le terminal.
- executez la commande : doxygen -g
- executez la commande : doxygen Doxyfile
